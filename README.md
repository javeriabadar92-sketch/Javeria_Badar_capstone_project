# ProjectPilot AI

**From idea to implementation—all in one AI-powered workspace.**

## Overview

ProjectPilot AI is a responsive web application that helps Software Engineering students and beginner developers transform a simple project idea into a structured software project plan.

Instead of switching between ChatGPT, Notion, Trello, Google Docs, and Figma, users can manage everything inside one AI-powered workspace.

The application focuses on project planning, documentation, UI inspiration, and task management — not code generation.

## Status

🚧 In development — Capstone project for Frontend AI Engineering internship track.

## Tech Stack

- **Frontend:** React, TypeScript, Vite
- **Styling:** Tailwind CSS
- **Routing:** React Router
- **AI:** OpenAI API
- **Deployment:** Vercel (planned)
